我所學到的知識
1.CSS Border
(1)width(線的粗細)
(2)style(線的特色)
(3)color(線的顏色)
(4)top、right、bottom、left(線的上、右、下、左)
(5)radius(線的彎曲度)

2.CSS Font
(1)family(文字的特色)
(2)style(文字傾斜的程度)
(3)size(文字的大小)
(4)weight(文字的粗細程度)
(5)variant(把所有小寫的字母轉換成大寫的字母)

3.CSS Text
(1)color(字的顏色)
(2)align(文字向哪邊靠齊)
(3)decoration(文字的上線、中線和底線)
(4)transform(所有英文字母大寫或小寫、每個單字的第一個字母大寫)
(5)indent(每段文字開頭時留多少空間)
(6)letter-spacing(每個字母之間的差距)
(7)line-height(每行之間的差距)
(8)direction(整個英文句子順序顛倒)
(9)word-spacing(每個單字之間的差距)
(10)text-shadow(字的影子)

4.HTML Table
(1)rowspan(直排格子的合併)
(2)colspan(橫排格子的合併)
